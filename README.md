💻 VS Code Portfolio

A developer portfolio with a Visual Studio Code-inspired interface, designed to provide an interactive way to explore professional experience, technical skills, projects, certifications, and contact information.

🚀 Overview

This portfolio delivers a familiar development environment by recreating the look and feel of Visual Studio Code. Instead of traditional web navigation, content is organized into files and folders, allowing visitors to browse information through an interface commonly used by developers.

✨ Features

⚙️ Getting Started

Clone the repository:

git clone https://github.com/<username>/<repository>.git

Navigate to the project:

cd <repository>

Install dependencies:

npm install

Run the development server:

npm run dev

Build for production:

npm run build
